# Shreeji Water Treatment

A modern, responsive website for **Shreeji Water Treatment**, showcasing electronic water softeners (descalers) for residential, commercial, and industrial applications.

Built with **Next.js (App Router)** and **Tailwind CSS**, the site is optimized for performance, SEO, and mobile devices.
